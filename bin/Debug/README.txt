Привет дорогой пользовате!!! Ты как понял я разрабочик этой программы!!!
Я расскажу как пользоваться программой!!!
Программа нужна для создания своего файл и записив него как в ежедневника свои данные 
(идеи или мысли, а может что нужно быстро записать!!!)
После открытия ты видешь не большой интерфейс с не большим функционалом!!!
Для работы тебе надо выбрать файл, там же есть спецальная кнопка с соответствующей кнопкой!
После выбора файла, ты можешь записать данные в поле на интерфейсе, и после кнопки записать ты запишишь премеком в файл котрый ты выбрал 
Так же ты можешь создать сам файл с помошь кнопки на  меню File -> Create file -> дать название и сохранить.
- Кнопка рандомные числа записывают как и очивидно рандомные числа от 0 до 10, с верху выпадающий список, при выборе числа
вы выберите длину рандомных цифр(попробуйте и всё паймёте)
- Кнопка "Очистить поле" очищеает поле на интерфейсе
"Меню"
- "Help..." показывает кто разработал программу
- "Очистить файл" очищает файл который вы выбрали или создал, то есть делает его пустым полностью.
- "Справка" ну тут понятно, это то что вы сейчас читаете (капитан очивидность!)))
- "Очистить путь файла" очищает путь который вы выберали или создавали для файла, то есть вы не чиго не сможете записать, пока по новой 
не создадите или аноалогично не выберати файл!!!
Спасибо за внимание, думаю что программа вам понравилась!!!
